# JJS-Visual-Character-Maker
The JJS Visual Character Maker (JJSVCM for short) is a small project I made that allows people to make JJS characters in a Scratch like enviorment, with the ability to import and export character codes.
This project is completely open-source and should try to be improved when a new feature of JJS comes out.
